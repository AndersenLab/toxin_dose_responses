# Natural Variation in Toxin Susceptibility Among Diverse _C. elegans_

A one-stop-shop for ongoing analyses of image-based toxin dose-responses assays in _C. elegans_ using CellProfiler and easyXpress.

## Required Software
1. [R-v3.4.1](https://www.r-project.org/)
1. [R/workflowr](https://github.com/jdblischak/workflowr)
1. [R/easyXpress](https://github.com/AndersenLab/easyXpress)
1. [R/tidyverse](https://www.tidyverse.org/)
1. [R/drc](https://cran.r-project.org/web/packages/drc/index.html)
1. [R/ddpcr](https://www.rdocumentation.org/packages/ddpcr/versions/1.13)
